My Projects

Weapon Detection in Real time using CCTV

The project focuses on detecting weapons in crowded areas using live CCTV footage to prevent potential crimes. Unlike existing models that analyze recorded videos, this system utilizes real-time streaming, providing faster detection and response. The process involves capturing live footage, converting it into frames, performing data preprocessing, and extracting features to identify weapons and individuals carrying them. Upon detection, a red bounding box is drawn around the individual, and an alert is sent to the nearest control center for immediate action.

Fraud detection in Banking Transactions

The fraud detection system aims to identify fraudulent activity in real time, halting fraudulent transactions as soon as they occur. It focuses on minimizing false positives to ensure valid transactions aren't incorrectly flagged, preserving the user experience. The system is designed to be scalable and flexible to handle high transaction volumes and adapt to new fraud tactics. Additionally, using explainable AI algorithms promotes transparency by providing clear reasons for flagged transactions, which enhances user trust and supports regulatory compliance.
